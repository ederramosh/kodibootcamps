¿Qué vas a aprender?

Este bootcamp está diseñado para principiantes y programadores que buscan llevar sus habilidades al siguiente nivel. Cubriremos tecnologías clave en el desarrollo web moderno.

Temario del Bootcamp

Módulo 1: Fundamentos de Desarrollo Web

  • HTML, CSS y JavaScript
  • Control de versiones con Git y GitHub
  • Conceptos de desarrollo Frontend

Módulo 2: Frameworks y Librerías

  • Introducción a React y Next.js
  • Manejo de estado con Zustand
  • Estilización con Tailwind CSS

Módulo 3: Backend y Bases de Datos

  • Autenticación con Supabase
  • Manejo de Bases de Datos SQL
  • Despliegue y hosting de aplicaciones
